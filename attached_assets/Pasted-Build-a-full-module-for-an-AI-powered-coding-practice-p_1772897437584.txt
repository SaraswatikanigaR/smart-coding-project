

The module will integrate with an existing Monaco code editor that another developer built.

My module starts after the code editor and handles:

Problems page
Problem descriptions
Test case evaluation
Static analysis
AI feedback

DESCRIPTION TAB

Show:

difficulty badge
topic badge

problem description

examples

constraints

Example:

Given an integer n, return a string array answer where answer[i] is:

"FizzBuzz" if divisible by 3 and 5
"Fizz" if divisible by 3
"Buzz" if divisible by 5
otherwise the number.

Example Input:
n = 3

Output:
["1","2","Fizz"]

Ask a conceptual question.
